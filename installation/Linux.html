Ubuntu 18.04

A successful example is Ubuntu.

The working kernel shown in uname -a is 4.15.0-42-generic #45-Ubuntu.

The Nvidia related packages are: Nvidia-driver Version: 410.78 CUDA Version: 10.0

Gentoo

PyNUFF works with the package in Gentoo system: dev-util/nvidia-cuda-sdk-10.1.243, dev-util/nvidia-cuda-toolkit-10.1.243-r1 dev-util/intel-ocl-sdk-4.4.0.117-r1, and x11-drivers/nvidia-drivers-450.66.
